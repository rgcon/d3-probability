# Probability Theory with D3

This is a small standalone D3.js explainer for core probability ideas.

It includes:

- A sample-space view for event probability on a fair die
- A binomial distribution explorer with adjustable `n` and `p`
- A Bernoulli simulation showing the law of large numbers

## Run

Open `index.html` in a browser.

If you prefer serving it locally, any static file server will work.

## Files

- `index.html` contains the page structure
- `styles.css` defines the visual design and layout
- `script.js` builds the D3 visualizations and interactivity
